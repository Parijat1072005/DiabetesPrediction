                       DiabetesPrediction
This repository contains two python files having codes which predicts the diabetes in people.

                How to run them in your code editor
Make a directory for this project, and open vs code.
open the terminal in vs code and follow the steps below.



>>cd <path_to_the_project_directory>



>>git clone https://github.com/Parijat1072005/DiabetesPrediction



>>cd DiabetesPrediction



>>python -m venv venv



>>.\venv\Scripts\activate



>>pip install -r requirements.txt


Ignore the commit message in the files since there are .ipynb files having outputs so they keep updating and hence the message to commit stays so either just ignore that or remove all the outputs from the file.




You can now directly run the whole file or run each cell one by one as per your choice just after selection right interpreter of python.
